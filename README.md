# 🚀 Taller 3 - Proyecto Web A.PRO

Proyecto desarrollado con **HTML5** y **Bootstrap 5**, como parte de las actividades de formación del programa **ADSO - SENA**.

---

## 📋 Descripción
Sitio web informativo para la empresa **A.PRO**, dedicada a brindar asesoría y soluciones para emprendedores. Incluye varias secciones organizadas:
- Página de inicio
- Listado de empleados/asesores
- Formularios de registro e inicio de sesión
- Guía de uso de Git y GitHub
- Diseño responsivo adaptado a móviles, tabletas y computadoras

---

## 🛠️ Tecnologías utilizadas
- **HTML5**
- **Bootstrap 5.3.8**
- **Bootstrap Icons**
- **Git & GitHub**
